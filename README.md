# 防府市学習アシストアプリ (Study Assist App)

## 概要
学習をサポートするためのWebアプリケーションです。ポモドーロタイマー、学習カレンダー、テストアーカイブなどの機能を提供します。

## 機能
- **ポモドーロタイマー**: 25分の集中と5分の休憩を管理
- **カレンダー**: 学習計画の作成と確認 (開発中)
- **テストアーカイブ**: 過去のテスト結果の保存と閲覧 (開発中)

## 技術スタック
- React
- Vite
- Tailwind CSS (想定)
- Lucide React (アイコン)
- React Router (ルーティング)

## セットアップ
1. 依存関係のインストール
   ```bash
   npm install
   ```

2. 開発サーバーの起動
   ```bash
   npm run dev
   ```

## ディレクトリ構造
- `src/components`: UIコンポーネント
- `src/services`: 外部API連携など
- `src/utils`: ユーティリティ関数
- `config`: 設定ファイル
- `data`: データファイル
